# HR Analytics Dashboard using Excel

## Project Overview
This project focuses on analyzing HR data using Microsoft Excel to uncover workforce trends, employee attrition patterns, and key human resource metrics. The interactive dashboard helps HR professionals make data-driven decisions by providing insights into employee demographics, job roles, attrition rates, and overall workforce performance.

## Objectives
- Analyze employee attrition and retention trends.
- Monitor key HR performance indicators (KPIs).
- Identify patterns across departments, job roles, and demographics.
- Support strategic HR decision-making through data visualization.

## Tools & Features Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Excel Formulas
- Interactive Dashboard Design

## Key Insights
- Employee attrition analysis by department and job role.
- Workforce distribution based on age, gender, and education.
- Analysis of employee performance and satisfaction metrics.
- Identification of factors influencing employee turnover.

## Dashboard Features
- Dynamic KPI cards
- Interactive slicers for filtering
- Department-wise analysis
- Attrition trend visualization
- Employee demographic analysis

This project demonstrates how Excel can be leveraged as a powerful Business Intelligence tool for HR analytics and workforce management.
